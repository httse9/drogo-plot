Figure A. Top row: Three Point Maze environments from gymnasium robotics, adapted to include low-reward regions. Green indicates agent position, red circles indicates goal positions, while red squares indicates low-reward regions. Similar to our grid world experiments, the agent receives -1 reward unless it visits low-reward regions (-20 reward), until it reaches the goal. A key distinction between these environments and earlier grid world environemnts is that the state space is now continuous, and it is not possible to compute the eigenvectors using eigendecomposition. Middle row: the principal eigenvector of the DR learned using DROGO. Bottom row: The principal eigenvector of the SR learned using ALLO (Diego et al., 2024).
